## instruções GIT

| Instrução | Descrição | 
|-|-|
|git init|iniciar o monitoramento do projeto|
|git status|verifica o estado dos arquivos presentes|
|git add|Preapra os arquivos para uma nova versão|
|git commit -m "mensagem"| Cria uma nova versão.|
